# 00_PROJECT_BRIEF.md

## 한 줄 정의
Checkmate는 친구끼리 비공개 인증방을 만들고, 가상 포인트를 걸고, 서로의 인증을 확인하며, 목표 인증률 달성자끼리 포인트를 자동 정산하는 서비스다.

## 문제
카카오톡 인증방은 시작은 쉽지만 관리가 어렵다.
- 누가 인증했는지 보기 어렵다.
- 사진/기록이 채팅에 묻힌다.
- 인증률 계산과 약속금 정산이 수동이다.
- 결과를 예쁘게 공유하기 어렵다.

## 해결
- 초대 링크 기반 비공개 방
- 가상 포인트 예치
- 인원 충족 + 전원 예치 후 방장 시작
- 시작 다음 날부터 미션 시작
- 텍스트 또는 이미지 인증
- 친구 확인 시 인증 완료
- 목표 인증률 기준 자동 정산
- 결과 카드 데이터 제공

## MVP 제외
공개방, 알림, WebSocket, S3, 실제 결제, PG, 에스크로, Redis, Batch, 관리자, AI, 서버 이미지 생성.
